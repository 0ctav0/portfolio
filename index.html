Minesweeper DirectX 11
Github: D3D11Minesweeper

I strugled a lot. There's a lot of new topics. More of that is a lot of outdated information. An old DirectX 11 functions got deprecated, Effects (FX) is also deprecated. I used DirectXTK since it has functions that replace deprecated old ones and it has really easy to use many of "vanila" DirectX features.

I briefly got to know Windows API programming, standard Windows message queue loop. DirectX 11 setup code, uploading shaders, different kind of buffers into memory, game loop.

I used a texture atlas for rendering many sprites for the game.
